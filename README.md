# Pharma-Sales-Inventory-ETL-Pipeline
This ETL pipeline consolidates pharmaceutical sales, inventory, and drug metadata into a Postgres data warehouse. It cleans, joins, and computes KPIs like revenue and stockout risk, orchestrated via Airflow. A Metabase dashboard surfaces key business insights, with AI assisting at every stage — from data generation to SQL and storytelling.
